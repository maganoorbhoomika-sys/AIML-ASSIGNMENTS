

Mini Project Proposal: Student Performance Predictor

1. Title:Student Performance Predictor

2. Problem Statement

Many students find it difficult to estimate their exam performance based on their preparation. This project aims to build a system that predicts a student’s exam score using factors like study hours, attendance, and previous marks. This helps students understand their performance level and improve accordingly.


3. Dataset

The dataset will include the following features:

Study Hours
Attendance Percentage
Previous Exam Marks

The dataset can be:

Collected manually (survey/class data), or
Taken from publicly available student datasets

4. Algorithm Used

Linear Regression-(to predict marks as a continuous value)
Decision Tree- (to classify performance as Pass/Fail)

5. Methodology / Steps

1. Data Collection
   Gather student-related data (study hours, attendance, marks)

2. Data Preprocessing

    Handle missing values
    Convert data into proper format
    Normalize if needed

3. Model Training

    Train Linear Regression model for score prediction
    Train Decision Tree for classification

4. Prediction

    Input new student data
    Predict exam score
    Classify as Pass/Fail



6. Expected Output

 Predicted exam score (numerical value)
 Performance category:

Pass (if score ≥ threshold)
 Fail (if score < threshold)



7. Applications

Helps students plan studies better
Assists teachers in identifying weak students
 Useful for academic performance analysis
